安装数据库操作类

  • 数据库操作类的安装跟Action中一样,安装到Mod中去,我写了一个Mysql的数据库操作类,实现了单表连贯操作(当然你可以安装别的,如果合适的话)
  • 下载:github:https://github.com/hit9/mysql-db/blob/master/db.class.php
  • 或者: git clone git@github.com:hit9/mysql-db.git
  • 安装:
    • 把db.class.php放到Core/Mod/db/下并在Core/Mod/mod.config.php的数组中加入'db'
    • 配置项:DB_HOST,DB_CHAR_SET,DB_NAME,DB_USER,DB_PASSWD,TABLE_PREFIX,QUERY_ERROR
    • 一个sample配置(Core/Config/config.php中): 
      		 /*
		 *db配置
		 */
		define('DB_HOST','localhost');//主机
		define('DB_NAME','testcore');//数据库名字
		define('DB_USER','root');//数据库管理用户名
		define('DB_PASSWD','*****');//数据库管理密码
		define('DB_CHAR_SET','utf8');// 数据库连接编码
		define('TABLE_PREFIX','t_');//数据库表名前缀
		define('QUERY_ERROR',true);//数据库查询报错开关,开发时建议开,生产环境下关闭
  • 理论上这个数据库操作类不支持安装为Mod的父类
