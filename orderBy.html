orderBy()

  • 类似于 limit()我们也可以使用where来实现orderBy 
    	public function orderBy()
	{
		$user=M('user');
		$re_arr=$user->db->where('id>0 ORDER BY id')->find();
		print_r($re_arr);
	}
  • 更简洁的使用orderBy函数: 
    	public function orderBy()
	{
		$user=M('user');
		$re_arr=$user->db->where('id>0')->orderBy('id')->find();
		print_r($re_arr);
	}
  • 如果想要反序,使用orderBy('id DESC')
  • orderBy()传入一个合法的字符串,输出一个数据对象
