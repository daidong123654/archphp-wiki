update()

  • update($id='',$arr=array());函数参数是主键的一个值或者使用前面指定的查询条件,返回bool
  • Sample code

使用对象方式:

  • 1.按照主键更新 
    	public function update()
	{
		$user=M('user');
		$user->db->name='小红';
		$user->db->email='xiaohong@gmail.com';
		$user->db->update(1);
	}
  • 2.按照条件更新 
    	public function update()
	{
		$user=M('user');
		$user->db->name='小红';
		$user->db->email='xiaohong@gmail.com';
		$user->db->where('name="小明"')->update();
	}
    或者使用数组更新条件: 
    	public function update()
	{
		$user=M('user');
		$user->db->name='小红';
		$user->db->email='xiaohong@gmail.com';
		$condition['name']='小明';
		$condition['email']='xiaoming@gmail.com';
		$user->db->where($condition)->update();
	}
