limit()

  • 实现查询数据的limit可以使用 where 
    	public function limit()
	{
		$user=M('user');
		$re_arr=$user->db->where('id>0 limit 1')->find();
		print_r($re_arr);
	}
  • 但数据库操作类内置的limit()更直观 
    	public function limit()
	{
		$user=M('user');
		$re_arr=$user->db->where('id>0')->limit(1)->find();
		print_r($re_arr);
	}
  • 一般地limit函数这么调用 : limit('1,2') (从第一条开始读取2个数据).返回一个数据对象
