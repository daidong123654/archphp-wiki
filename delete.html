delete()

  • delete($id='').函数参数是主键的一个值或者采用前面指定的查询条件,返回bool
  • Sample code
  • 按照主键删除一个记录: 
    	public function delete()
	{
		$user=M('user');
		$user->db->delete(1);
	}
    按照条件来删除一些数据: 
    	public function delete()
	{
		$user=M('user');
		$condition['name']='小红';
		$user->db->where($condition)->delete();
	}
