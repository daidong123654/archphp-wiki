add()

  1. 假如我们已经建立好数据库testcore,并且已经建立了表t_user (字段id,name,email).每个表名(不含前缀)对应一个同名的Mod
  2. 添加数据到数据库add():对象方式
    • 首先建立一个Mod类:Myapp/Mod/user.class.php: class UserMod extends Mod
    • 然后在IndexAction中写测试: 
      		<?php
		class IndexAction extends Action
		{
			public function add()
			{
				$user=M('user');
				$user->db->name='小明';
				$user->db->email='xiaoming@gmail.com';
				$user->db->add();
			}
		}
      • 访问localhost/Myapp/index.php?a=add会添加这个数据到数据库
  3. 添加数据到数据库:数组方式 
    		public function add()
		{
			$user=M('user');
			$data['name']='hh';
			$data['email']='hh@qq.com';
			$user->db->add($data);
		}
