post()

  • 雷同于get().post($varname='')接受的参数是输入数据的属性名字或不接受参数.当接收一个数据的时候返回这个数据,当接收多个数据的时候返回一个一维数组
  • post()用于检查输入的数据是否是POST数据
  • Sample Code: 新建一个表单sample.html: 
    	<html>
	<body>
		<form action="?a=foo" method="post" accept-charset="utf-8">
			<p>
				name:<input type="text" name="name" value="">
			</p>	
			<p>
				email:<input type="text" name="email" value="">
			</p>
			<p><input type="submit" value="Continue &rarr;"></p>
		</form>
	</body>
	</html>
    然后在IndexAction中加个函数来显示这个表单,比如我使用的模板引擎是tpl 
    	public function form()
	{
		$this->tpl->display('sample');
	}
    然后在IndexAction中加入foo函数接收数据: 
    	public function foo()
	{
		echo $this->input->post('name');
		echo $this->input->post('email');
	}
    接下来访问localhost/Myapp/?a=form填写表单测试下~.
