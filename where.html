where()

  • where()方法接受字符串和数组参数,返回一个数据对象
  • 可行的sample code: 
      	public function where()
	{
		//字符串参数和数组参数例子
		$user=M('user');
		$re_arr1=$user->db->where('id=1 AND name="小明"')->find();
		$condition['id']=1;
		$condition['name']='小明';
		$re_arr2=$user->db->where($condition)->find();
		$re_arr3=$user->db->where('id=1')->where('name="小明"')->find();
		print_r($re_arr1);//三个数组打印结果一致
		print_r($re_arr2);
		print_r($re_arr3);
	}
  • 通过code可以看出,where支持字符串条件,数组条件,支持一串长龙式的where连贯
