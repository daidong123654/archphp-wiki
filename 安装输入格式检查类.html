安装输入格式检查类

  1. 简介:格式检查类负责输入数据的格式检查.作为Action的一个属性安装
  2. 下载:cd Core/Action/
    • git clone git@github.com:hit9/input.git
  3. 编辑Core/Action/action.config.php,在数组中添加'input'
  4. 配置项:INPUT_ERROR_ECHO(设置为true的时候,遇到格式不符合数据echo错误名字并结束PHP脚本) 在Core/Config/config.php中添加: 
    	/*
	 *input配置
	 */
	define('INPUT_ERROR_ECHO',true);//回显格式检查错误并die
