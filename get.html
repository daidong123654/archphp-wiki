get()

  • 函数get($varname='');接受参数是表单input框的name或者不接收参数.当get()只接收一个输入数据的时候返回这个数据;当接收多个输入数据的时候返回一个一维数组
  • get()函数用来检测接收的数据是否是GET类型
  • Sample Code: 
    	<?php
	class IndexAction extends Action
	{
		public function foo()
		{
			echo $this->input->get('test');
		}
	}
    • 访问localhost/Myapp/?a=foo&test=你好,会看到顺利接收了test数据:你好
    • 如果我们访问localhost/Myapp?a=foo,会看到显示了test.所以,错误的抛出是输入的input的name(属性名字)
