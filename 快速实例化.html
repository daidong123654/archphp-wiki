框架全局函数:快速实例化

全局函数的使用

  • 在框架Core/Comm/function.php文件中定义了三个全局函数A(),M()和Addon()
  • A() 快速实例化一个项目中的Action类并返回对象
  • M() 快速实例化一个项目中的Mod类并返回对象
  • Addon() 快速实例化一个项目中的Addon类并返回对象
  • 下面是一个例子:

文件:Action/index.class.php 

<?php
class IndexAction
{
	public function index()
	{
		$user=M('user');//快速实例化UserMod
		$user->foo();//执行类UserMod的函数foo()
	}
}
文件:Mod/user.class.php 
<?php
/**
 *UserMod
 **/
class UserMod extends Mod
{
	public function foo()	
	{
		echo 'hello world';
	}
}

  • 上例中访问localhost/Myapp会看到小写的hello world
  • Core/Comm/function.php中定义的函数可以在程序中的任何位置使用
