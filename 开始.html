开始一个项目

1.Hello World

首先要求环境:PHP 5.3+ (以Linux为例,win下可看注释来操作)

  1. mkdir Myapp #新建一个项目目录比如Myapp
  2. cd Myapp #转到新建的目录中去
  3. git clone git@github.com:hit9/Core.git #下载框架到目录下
  4. touch index.php #新建入口文件index.php
  5. vim index.php #编辑入口文件如下 = 
    	<?php
	/*
	 *项目入口文件
	 */
	header("Content-type:text/html;charset=utf-8");//设置全局项目编码utf8
	define('CORE_PATH','./');//定义框架相对入口文件的路径
	require_once(CORE_PATH.'Core/core.php');//包含框架核心文件
	Action::run();//执行分发请求
  6. mkdir Action Mod Addon #在项目目录下建立三个目录Action Mod Addon
  7. cd Action #转到Action目录下开始编写代码啦
  8. touch index.class.php #建立一个action类
  9. vim index.class.php #编写如下代码 
    	<?php
	/**
	*Hello World !
	**/
	class IndexAction
	{
		public function index()
		{
			echo 'Hello World!';
		}
	}
  10. 访问localhost/Myapp/?c=index&a=index 或者直接访问localhost/Myapp
  11. 至此我们已经完成了一个Hello World

2.目录结构

  • 从上面的Hello World可以看出目录结构类似这样:
  • Action/ Mod/ Addon/ Core/ index.php (项目目录下)
  • 如果不想把核心框架放在项目目录下,请修改入口文件中的CORE_PATH的定义
